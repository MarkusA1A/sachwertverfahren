# Sachwertverfahren – Einfamilienhaus Wertermittlung

Eine interaktive Web-App zur schnellen Berechnung des Sachwertes von Einfamilienhäusern nach deutschem Standard.

## Features

✅ **Gebäudewert-Berechnung**
- Herstellungskosten basiert
- Alterswertminderung automatisch
- Wertverhältnis-Anpassung

✅ **Grundstückswert-Berechnung**
- Bodenrichtwert integriert
- Makler- & Nebenkosten abzugsfähig

✅ **Sofortiges Ergebnis**
- Keine Installation nötig
- Moderne Web-App
- Offline-ready

## Nutzung

1. Öffne die App: [sachwertverfahren.html](https://markusa1a.github.io/sachwertverfahren/)
2. Füll deine Werte ein:
   - **Wohnfläche** (m²) – nach WoFlV
   - **Herstellungskosten** (€/m²) – aktuelle Baukosten
   - **Alter** (Jahre) – seit Fertigstellung
   - **Gesamtlebensdauer** (Jahre) – typisch 70–100 Jahre
   - **Grundstücksfläche** (m²)
   - **Bodenrichtwert** (€/m²) – vom Gutachterausschuss
3. Berechnen klicken → Sachwert anzeigen

## Datenquellen

- **Bodenrichtwert:** Gutachterausschuss deiner Stadt/Gemeinde
- **Herstellungskosten:** Deutsche Preisindices für Bauinvestitionen (aktuell ~1.800–2.500 €/m²)
- **Lebensdauer:** Nach ImmoWertV typisch 70–100 Jahre

## ⚠️ Hinweis

Diese App ist ein **Schätzwerkzeug**. Für offizielle Wertermittlungen (Finanzamt, Bank, Gutachten) konsultiere einen zertifizierten Gutachter. Der Sachwert ist oft nur eine von mehreren Bewertungsmethoden (Vergleichswert, Ertragswert).

## Technologie

- Vanilla HTML5/CSS3/JavaScript
- Keine Abhängigkeiten
- Mobile-responsive
- Browser-basiert (keine Server-API nötig)
